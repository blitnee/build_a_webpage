HTML

What is HTML?

HTML stands for Hyper Text Markup Language. It is a set of syntax that allows videos, text, images, and music to exist on a webpage. HTML creates everything from text content and markup to links to other references and links to other pages.

What is an element?

Elements specify sections in code and determine the organization of an HTML document.

What is a tag?

A tag is a sort-of key starting or ending an element in HTML. Tags are most often created through an opening and closing structure. This is an example of standard tag structure:

<p>stuff</p>


Some types of tags do not need to be closed. This is called a void tag. Void tags are represented like this:

other stuff<br>

What are attributes?

An attribute typically goes inside of a tag and gives a value to the noted element. Attributes always hold a value. An attribute is used to classify and modify an HTML element in some way. This is an example of how you might use an attribute:

<tag attr="value">contents</tag>

What do 'inline' and 'block' mean?

An inline tag is an item that that is applied in the line of code. A <br> is an example of an inline element. It creates a break in the line of text. A block tag is an item that creates a separate block of text. <p> is an example of a block tag.

What is the difference between 'span' and 'div'?

A <span> is an inline element whereas a <div> is a block element. This means that when you use <div> in a line of text, it will create a separate block with the text which it includes.

How do I structure an HTML document?

Below is an example of how to structure an HTML Document:

<!DOCTYPE html>
<html>
     <head>
           <title></title>
     </head>
     <body>
           <p>
           </p>
     </body>
</html>

CSS

What is CSS?

CSS stands for Cascading Style Sheets. A style sheet is used to apply a styled look and formatting to elements in an HTML document. CSS can be described as the rules for a webpage.

How does CSS manage HTML?

CSS manages HTML elements through selectors and declarations. A selector is a title attached to an element that assigns the element to a class or ID. An ID is a unique identifier that is applied to a particular element in the HTML document. A class refers to a group of specified elements in HTML which are intended to be styled together. A declaration is a statement or rule that can be applied to elements in HTML through tags, class, or id. Declarations are the rules that element properties follow to create a style. To summarize, declarations are lists of properties and assigned values under a selector which create the style of an HTML document.

How do I apply CSS?

There are three ways to apply CSS to your HTML document.
  1. You can write CSS in the <head> section of your HTML. This method works best for small projects.
  2. You can link your HTML to a separate CSS file. This is method works well for larger projects.
  3. You can write your CSS inline with your HTMl. However, this method is not recommended because you will need to duplicate a large amount of code.
